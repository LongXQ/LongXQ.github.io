openjdk的源码是用Mercurial版本工具管理的。我们以下载jdk11源码为例，介绍怎么下载。

1. 安装Mercurial

>>> pip install mercurial

2. 从仓库下载源码

>>> hg clone http://hg.openjdk.java.net/jdk-updates/jdk11u/

3. 切换到稳定分支

>>> hg update jdk-11.0.2-ga
